Orellius override PIN

Required when one Claude session needs to break another active session's tab/window lock. The first session to use a valid PIN claims it - other sessions are rejected until you Clear Binding or Rotate.

Set custom PIN
Pass to a destructive Orellius tool as override_pin:"<digits>". Affected tools: browser_lock, browser_unlock, tabs_close_mcp, session_end. Setting/rotating the PIN auto-clears any existing session binding.

Close Orellius windows
Close unused: closes windows belonging to dead Claude sessions; live sessions keep their tabs.
Close all: closes every Orellius window. Connected MCP clients stay - their next browser call auto-creates a new window.
Shell equivalents: curl -X POST :18766/admin/close-unused · :18766/admin/shutdown
